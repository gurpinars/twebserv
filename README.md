# twebserv
Just supports GET method and transmits static page. 
###Compile
```bash
make
```
###Usage
```bash
./twebserv -p [port] /path/root/
```
If you want to re-compile
```bash 
make clean
make
```
for help
```bash
./twebserv -h
```
